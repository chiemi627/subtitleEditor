完了済み:
1) Scaffold project
   - ローカルで動く Node.js + Vite + React のワークスペースを作成

2) Video player UI
   - MP4 の読み込み、再生／一時停止、5秒巻き戻し／5秒早送り、シークバー（ミリ秒精度対応）

3) SRT import & parser
   - `src/utils/srt.ts` にパーサ（`parseSrt`, `stringifySrt`）を実装

4) Subtitle list UI
   - `src/components/SubtitlesList.tsx` に読み込み、表示、テキスト編集、開始/終了時刻編集を実装

5) Sync playback with subtitles
   - 再生中の字幕をハイライトして自動スクロール（開始時刻設定時はスクロールを遅延させる処理で入力がずれないよう調整）

6) Save edited SRT
   - 編集済み字幕を `.srt` としてダウンロード可能（`stringifySrt` によりミリ秒出力）

7) Quick-set start/end from video
   - 開始/終了入力横にアイコンボタンを追加し、動画の現在時刻（ミリ秒単位）を設定可能
   - 開始を設定した場合は直前の字幕の終了時刻を同一時刻に自動調整

8) Delete subtitle
   - 各字幕行に削除（ゴミ箱）ボタンを追加。削除は即時で親 state に反映。
